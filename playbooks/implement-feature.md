# Playbook: Implement Feature

1. Confirm acceptance criteria, constraints, and active thresholds from `../references/quality/team-tunables.md`.
2. Add or update tests for expected behavior (RED).
3. Implement minimum code to satisfy tests (GREEN).
4. Commit production-ready change.
5. Refactor for clarity and maintainability.
6. Commit refactor-only change.
7. Run pre-commit checklist before final push.
